Chastity Game (D8 to Start, Add Die at Month 2 and 3, Die Gains One Face Every Day)
===================================================================================
 
HOW TO RUN IT
- Just double-click "Chastity Game.html". It opens in your default web browser
  (Edge, Chrome, Firefox, etc.) and works fully offline. No installation needed.
 
HOW TO PLAY
- Each day, open Chastity Game.html and click "Roll for today" ONCE. This is on the
  honor system. The app does not check real calendar dates, it only counts clicks.
- Day 1 starts as a d8. The die gains one face every day, FOREVER. This
  never stops or freezes (day 2 = d9, day 3 = d10, and so on indefinitely).
- You start with 1 die. One more die is added every month for the first
  3 months: 2 dice starting day 31, 3 dice starting day 61. After that
  (day 91 onward), the dice count is FROZEN at 3. No more dice ever get
  added, no matter how long the game runs. Only the die's face count keeps
  climbing.
- This means there's a real, low-but-meaningful chance of a game stretching
  well past a year, since the escalation stops but the die keeps growing.
- You win the moment any single roll comes up a 1. Game over.
- Click "Start a new game" to play again.
 
MEMORY / PERSISTENCE
- Your progress is saved automatically inside this browser's local storage,
  tied to this exact file. It will still be there next time you open Chastity Game.html,
  even after closing the browser or restarting your computer.
- IMPORTANT: Don't move, rename, or copy this file to a different location
  after you've started playing. Your saved progress is tied to this file's
  exact path. If you move the folder, just move the WHOLE folder together
  (keep Chastity Game.html at the same relative path) and it should still work in
  most browsers, but the safest approach is to just leave the folder where it is.
 
RESETTING
- Click "Reset game" at any time to wipe progress and start over.
 
SIMULATING OUTCOMES
- Scroll down to the "Simulate outcomes" box. Enter how many games you want
  to simulate (default 1000) and click "Run simulation". This is a sandbox;
  it does NOT touch your real, ongoing game above. Click it as many times as
  you like to see how the odds play out across many hypothetical
  games: average/median day of win, chance of ending within a week, month,
  3 months, or a year, plus a simple bar chart of outcomes.
